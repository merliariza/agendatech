# Agendatech
https://github.com/karen-12771/AGENDATECH.git

Agendatech es una aplicación web diseñada para gestionar citas, empleados, usuarios, inventario y productos dentro de un mismo sistema.  
Incluye un panel administrativo donde se controlan las citas, el personal y el inventario, así como una vista para clientes enfocada en la visualización de productos y el uso de un carrito de compras.  
El proyecto combina funcionalidades de administración interna y una experiencia básica de compra para el usuario final, permitiendo mantener el control del negocio desde un solo lugar.

## Tecnologías utilizadas

### Backend
- Node.js
- MySQL 
- express-session
- bcrypt

### Frontend
- HTML
- CSS
- JavaScript (Fetch API)
